IN-GAME.tech API

Here you have all the in-game.tech APIs

Faceit API

Nightbot command for https://in-game.tech/faceit/*faceit*

URL example: https://in-game.tech/faceit/IFreeze23

!addcom *commandname* $(eval const api = $(urlfetch json https://in-game.tech/faceit/*faceit* ); if (api.error) {'Error';} else { 'Faceit level: '+api['level']+ ' ELO: ' +api['elo']+ '. ELO needed for rankup/rankdown ' +api['rankup']+'/'+api['rankdown']})
